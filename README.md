In this game, you can play the built in maps and get the fastest time, or you can create your own maps and play on it. To make maps, follow the instructions down below.

Making maps:
1) You have to make a file with the extention ".txt" and edit it in an application.
2) To make maps, type in the tilenumbers in the file in the format you want it to display on the screen(e.g. 0 is typed at column 1 row 1 of the txt file and it will show rpthe tile of tilenumber 0 at screen column 1 screen row 1)and hava a space between every tilenumber.
